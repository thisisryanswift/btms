# BTMS - Better Tab & Session Manager

> 🧠 An AI-first reimagining of browser session management

## Project Status

🚧 **In Development** - Foundation setup in progress.

## What is this?

BTMS is a personal project to build a modern, AI-powered browser extension for managing tabs and sessions. Inspired by [Tab Session Manager](https://github.com/sienori/Tab-Session-Manager), but designed from the ground up with AI as a core feature.

## Project Status

🔬 **Research Phase** - Currently investigating existing solutions and planning AI integration.

## Key AI Features (Planned)

- 🏷️ **Smart Naming** - AI-generated meaningful session names
- 🔍 **Semantic Search** - "Find my React project from last week"
- 📊 **Session Summaries** - AI-generated overviews of what's in each session
- 🧹 **Duplicate Detection** - Find and merge duplicate tabs
- 🤖 **Proactive Suggestions** - "Open your work session?"

## Tech Stack (Proposed)

- **Extension**: Manifest V3, React, TypeScript
- **AI**: OpenAI API / Ollama for local LLM
- **Storage**: IndexedDB for sessions, browser.storage for settings
- **Build**: Vite or WXT
