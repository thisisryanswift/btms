# BTMS Chrome extension options panel bugs

The user demonstrates bugs in the BTMS Chrome extension options panel. Some options do not work, such as theme and language selection, and the interval setting for auto-save is not draggable. Some links on the about page go to a dead link.

## Steps to Reproduce

1. Go to the options panel for the BTMS extension.
2. Click on the theme in the appearance section to see that the theme doesn't change.
3. Select different languages in the language section to see that the language does not update.
4. Turn on automatic session naming and automatic session summaries to see that the session summaries do not display.
5. Turn on Auto-save and save in startup, notice that the labels do not make sense.
6. Try to drag the interval setting for autosave. Note that it does not drag.
7. Go to the about page and click the GitHub and Documentation links to see that they go to a dead link.
